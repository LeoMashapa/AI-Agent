# AI-Agent

This project uses Google Gemini 2.5 Flash as the reasoning engine and a collection of Python analytical tools to provide:

Natural language querying of query data
Business-focused insights and interpretations
Data quality analysis
Filtering and grouping
Percentage and share calculations
Ranking and trend analysis
Open queries monitoring
Team performance analytics

Instead of giving raw numbers, the application translates analytical results into meaningful business insights.
